이클립스에서 깃허브에 있는 프로젝트 불러오기

1. 깃허브 repository url 주소 복사
2. git repositories 탭에서 clone a repository... 되어 있는 아이콘(오른쪽위 녹색화살표 아이콘) 클릭
3. Clone URI 선택 후 Next 클릭
4. 1번에서 복사해 놓은 주소가 자동으로 입력됨(자동입력 안되면 다시 url 복사 후 맨 윗줄에 붙여넣기)
   (깃허브 계정과 토큰 입력 안되어 있으면 확인 후 입력)->next 클릭
5. 에러가 없는 프로젝트면(url이 올바른 주소면 에러 없음)->next
6. Directory가 본인 컴퓨터에 깃허브에서 가져올 프로젝트가 저장될 위치임
    ->Browse 버튼을 클릭해서 깃허브 프로젝트를 다운로드할 폴더를 지정->Finish
    (주의! 가능하면 깃허브에서 가져올 프로젝트만 따로 모아 놓는 폴더를 지정)
    (주의! 바탕화면이나 문서폴더 안됨, 폴더를 변경하지 않고 다운로드 받으면 사용자 폴더에 생성됨으로 주의!)
7. 주의사항! 깃허브에서 가져올 프로젝트의 이름과 동일한 프로젝트가 이미 본인 이클립스에 프로젝트 익스플로러에 존재하면 에러남(못불러옴 반드시 확인 필요!)
8. git repositories 탭에 불러오려는 깃허브 프로젝트 이름이 출력됨
9. File->import 메뉴 클릭
10 . Import 메뉴창에서 General->Existing Projects into Workspace 선택 후 next
11. Select root directory에 있는 Browse 버튼을 클릭하여 깃허브에서 가져온 프로젝트가 저장된 폴더(6번에서 지정한 폴더)를 선택 후 확인 누르고 에러 없는지 확인 후 Finish 클릭
12. 왼쪽의 패키지 익스플로러에 해당 프로젝트가 올라옴(만약 환경설정이나 에러가 있는 프로젝트의 경우에 프로젝트 폴더이름에 x표시 뜸)
